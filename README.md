# Installing the packages
npm install
// OR //
bun install

# Running the servers
## Node
node run dev
## Bun
bun run dev

# Don't forget to fill every .env variables with your own credentials to test the project on your side
## The variables would be :
GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET
MONGO_USERNAME
MONGO_PASSWORD
MONGO_HOSTNAME
JWT_SECRET_KEY
GATEWAY_PORT
BACKEND_PORT
TASKS_PORT
AUTH_PORT
